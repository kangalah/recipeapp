# RECIPE APP

# BY Joan Nanjala,Stellah Waithera,Ryan Munge and Ivine Kemuma.

# Description
* This is a Recipe web application that allows users to add recipe,edit and delete a recipe.A user can also search and view other recipes from the MyCookbook.io Api.

# Behaviour Driven Development.
With the application, a user is able to:
* View different recipes.
* see recipes posted by other people.
* submit a recipe.
* to rate a recipe on whether its easy or hard.
* comment on the different recipes.
* A user is able to login or register for an account if he/she does not have one.

# Installation requirements.
One needs to have the following to start work on the project on their local computer:
* Python3 whichever version
* Flask installed.
* Pip installed.
* Virtualenvironment.
* A text editor.
* Run pip install -r requirements.txt to install or requirements used. 

# Getting started.
* Clone the repository to your local machine.
* Ensure you have python3 installed on your computer.
* From the terminal navigate to the cloned project folder.
* Create a virtual environment and access the folder.


# Technologies used.
* Python3
* Bootstrap.
* Flask.
* Pip.
